# Cartório EBAC

Projeto desenvolvido durante o curso de Tecnologia da Informação da [EBAC](https://ebaconline.com.br/), com foco na prática de fundamentos de programação em **C**.

## Sobre o projeto

Este repositório reúne os exercícios e atividades desenvolvidos ao longo do curso, servindo como registro da evolução técnica e espaço para experimentação com a linguagem C.

## Tecnologias

- **C**
- **Git e GitHub**

## Como começar

Clone o repositório:

```bash
git clone https://github.com/devzeckk/Cartorioebac.git
cd Cartorioebac
```

Depois, abra os arquivos do projeto em sua IDE ou editor preferido e compile-os com um compilador C, como o GCC.

## Aprendizados

- Fundamentos da linguagem C
- Organização de código
- Compilação e execução de programas
- Versionamento com Git
- Publicação de projetos no GitHub

Em desenvolvimento, conforme o avanço do curso.
